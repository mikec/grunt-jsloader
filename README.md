grunt-jsloader
==============

Set up your project for asynchronous script loading

Installation
------------

Install grunt-jsloader using npm:

```
$ npm install grunt-jsloader
```

Then add this line to your project's *Gruntfile.js*:

```javascript
grunt.loadNpmTasks('grunt-jsloader');
```

Testing
-------

```
$ cd grunt-jsloader
$ npm test
```
